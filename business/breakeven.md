# Breakeven Analysis

## Cost per Active User (USD)
- **Compute**: $0.05/user/mo (assuming 10 min video renders/mo, $0.01/min compute)
- **Storage**: $0.01/user/mo (1GB storage/user, $0.001/GB/mo)
- **Bandwidth**: $0.02/user/mo (1GB bandwidth/user, $0.002/GB)
- **Total CPAU**: $0.08/user/mo

## Pricing Tiers
| Tier       | Price/Month | Features                                                                 |
|------------|-------------|--------------------------------------------------------------------------|
| Starter    | $19         | 5 video renders/mo, basic templates, 480p resolution                     |
| Professional| $49         | 20 video renders/mo, advanced templates, 1080p resolution, priority queue |
| Enterprise | $199        | Unlimited renders, custom templates, 4K resolution, API access           |

## Customer Acquisition Cost (CAC) Range
- **Low end**: $20/user (organic growth, referral programs)
- **High end**: $50/user (paid advertising, influencer partnerships)
- **Average CAC**: $35/user

## Lifetime Value (LTV) Estimate
- **Average customer lifespan**: 12 months
- **Average revenue per user (ARPU)**: $39/mo (weighted average of tiers)
- **LTV**: $39/mo × 12 months × 25% gross margin = $117/user

## Break-even Users Count
- **Fixed Costs**: $5,000/mo (salaries, marketing, overhead)
- **Variable Costs**: $0.08/user/mo
- **Break-even Users**: Fixed Costs / (ARPU - Variable Costs) = $5,000 / ($39 - $0.08) ≈ 129 users

## Path to $10K MRR
| Tier       | Users Needed | Total Users |
|------------|--------------|-------------|
| Starter    | 526          | 526         |
| Professional| 204          | 204         |
| Enterprise | 51           | 51          |

**Note**: The path to $10K MRR can be achieved by a combination of users across different tiers. For example, 200 Starter users, 100 Professional users, and 20 Enterprise users would generate approximately $10,100 MRR.
